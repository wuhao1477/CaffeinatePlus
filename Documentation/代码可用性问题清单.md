### 2. 缺失的编译器桥接头文件

**问题**: CommonCrypto 在 Swift 中需要桥接头文件

**影响**: `LicenseService.swift` 中的 `CCHmac` 无法编译

**解决方案**:

创建 `CaffeinatePlus-Bridging-Header.h`:

```objective-c
#import <CommonCrypto/CommonCrypto.h>
```

或使用纯 Swift 方案（推荐）：

```swift
import CryptoKit

private func computeHMAC(for input: String) -> Data {
    let key = SymmetricKey(data: Self.HMAC_SECRET.data(using: .utf8)!)
    let signature = HMAC<SHA256>.authenticationCode(
        for: input.data(using: .utf8)!,
        using: key
    )
    return Data(signature)
}
```

### 5. AudioService 缺少异步处理

**问题**: `AudioService.startRouting()` 和 `stopRouting()` 是同步的，可能阻塞主线程

**位置**: `AudioService.swift:44-94`

**改进方案**:

```swift
func startRouting() async throws {
    guard !isRouting else { return }
    
    // 在后台线程执行
    return try await withCheckedThrowingContinuation { continuation in
        DispatchQueue.global(qos: .userInitiated).async {
            do {
                // 执行音频路由设置
                try self.performRouting()
                
                DispatchQueue.main.async {
                    self.isRouting = true
                    continuation.resume()
                }
            } catch {
                continuation.resume(throwing: error)
            }
        }
    }
}
```

### 6. SystemMonitorService CPU 计算不准确

**问题**: CPU 使用率计算只在第一次调用时正确，后续调用需要差值计算

**位置**: `SystemMonitorService.swift:78-116`

**当前实现**: 直接读取累计值
**问题**: 返回的是自启动以来的累计值，不是实时使用率

**正确实现**:

```swift
private var lastCPUTicks: [UInt64] = []

private func refreshCPUUsage() {
    // ... 获取 CPU info ...
    
    var currentTicks: [UInt64] = []
    for i in 0..<Int(numCPUs) {
        let offset = Int(CPU_STATE_MAX) * i
        let user = UInt64(info[offset + Int(CPU_STATE_USER)])
        let system = UInt64(info[offset + Int(CPU_STATE_SYSTEM)])
        let idle = UInt64(info[offset + Int(CPU_STATE_IDLE)])
        let nice = UInt64(info[offset + Int(CPU_STATE_NICE)])
        
        currentTicks.append(contentsOf: [user, system, idle, nice])
    }
    
    if !lastCPUTicks.isEmpty {
        // 计算差值
        var totalUsage: Double = 0
        for i in stride(from: 0, to: currentTicks.count, by: 4) {
            let userDelta = Double(currentTicks[i] - lastCPUTicks[i])
            let systemDelta = Double(currentTicks[i+1] - lastCPUTicks[i+1])
            let idleDelta = Double(currentTicks[i+2] - lastCPUTicks[i+2])
            let niceDelta = Double(currentTicks[i+3] - lastCPUTicks[i+3])
            
            let total = userDelta + systemDelta + idleDelta + niceDelta
            if total > 0 {
                totalUsage += (userDelta + systemDelta + niceDelta) / total
            }
        }
        
        cpuUsage = totalUsage / Double(numCPUs) * 100.0
    }
    
    lastCPUTicks = currentTicks
}
```

### 7. HotkeyService 缺少权限检查

**问题**: 没有检查辅助功能权限，导致快捷键可能静默失败

**位置**: `HotkeyService.swift:38-55`

**改进**:

```swift
init(onToggle: (() -> Void)? = nil) {
    self.onToggle = onToggle
    
    // 检查辅助功能权限
    if checkAccessibilityPermission() {
        startMonitoring()
    } else {
        Logger.shared.warning("Accessibility permission required for global hotkey")
    }
}

private func checkAccessibilityPermission() -> Bool {
    let options: NSDictionary = [
        kAXTrustedCheckOptionPrompt.takeUnretainedValue(): true
    ]
    return AXIsProcessTrustedWithOptions(options)
}
```

### 8. Logger 日志文件无限增长

**问题**: 没有日志轮转机制，日志文件会无限增长

**位置**: `Logger.swift:51-66`

**改进方案**:

```swift
private func writeToFile(_ message: String, level: LogLevel) {
    // 检查文件大小
    if let attributes = try? FileManager.default.attributesOfItem(atPath: logFileURL.path),
       let fileSize = attributes[.size] as? UInt64,
       fileSize > 10_485_760 { // 10MB
        rotateLog()
    }
    
    // ... 原有写入逻辑 ...
}

private func rotateLog() {
    let timestamp = DateFormatter.localizedString(
        from: Date(),
        dateStyle: .short,
        timeStyle: .short
    ).replacingOccurrences(of: "/", with: "-")
    
    let archiveURL = logsDirectory
        .appendingPathComponent("caffeinate-plus-\(timestamp).log")
    
    try? FileManager.default.moveItem(at: logFileURL, to: archiveURL)
    
    // 清理旧日志（保留最近5个）
    cleanOldLogs()
}
```
